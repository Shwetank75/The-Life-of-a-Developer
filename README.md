# 💻 The Life of a Developer – Interactive Web Experience

An immersive, scroll-based storytelling website that showcases the journey of a developer—from writing the first line of code to debugging chaos and final deployment.

🔗 **Live Demo:**
http://dev-journey-experience.netlify.app/

---

## 🚀 Overview

This project is designed as an interactive narrative experience rather than a traditional static website. Users explore different stages of a developer’s life through smooth scroll animations, transitions, and interactive UI elements.

The goal is to create a cinematic and engaging experience inspired by modern Awwwards-style websites.

---

## ✨ Features

* 🎬 Scroll-based storytelling experience
* 🌊 Horizontal scrolling sections
* ⚡ Smooth GSAP animations and transitions
* 🎮 Interactive elements (bug fixing, navigation, UI interactions)
* 🌌 Three.js animated background for depth
* ⌨️ Typewriter and micro-interactions
* 📱 Fully responsive design (mobile + desktop)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* GSAP (ScrollTrigger)
* Three.js

---

## 🎯 Project Structure

```
index.html   → Main HTML structure
```

## 🧠 Concept

The project transforms a relatable developer journey into an engaging interactive story. Instead of simply presenting content, it allows users to experience:

* Learning phase
* Debugging struggles
* Deadline pressure
* Final deployment success
